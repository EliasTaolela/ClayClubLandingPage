# 🏺 Clay Club Landing Page

Clay Club is a modern **ASP.NET Core MVC web application** designed as a creative pottery studio landing page with full backend functionality.

This project combines **beautiful UI (CodePen-inspired design)** with **real-world backend systems** such as authentication, role management, and admin control.

It is built to demonstrate how a simple landing page can evolve into a **fully functional web platform**.

---

## ✨ Features

### 🎨 Frontend
- Modern responsive landing page
- Sticky navbar with scroll effects
- Smooth section animations (fade/slide)
- Reusable Razor partial components
- Clean UI inspired by creative studios

### 🔐 Authentication
- User registration & login
- Secure authentication using ASP.NET Identity
- Logout functionality

### 👑 Admin System
- Role-based authorization (Admin/User)
- Admin dashboard
- View all users
- Delete users
- Promote users to Admin
- View user-specific data

### 🧠 Backend Architecture
- Clean service layer (separation of concerns)
- Entity Framework Core for database
- SQL Server integration

---

## 🛠 Tech Stack

- ASP.NET Core MVC (.NET 6/7/8)
- Entity Framework Core
- SQL Server / LocalDB
- Razor Views
- JavaScript (Intersection Observer)
- CSS (custom styling)
